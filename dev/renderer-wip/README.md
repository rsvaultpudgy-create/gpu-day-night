# Work-in-progress: from-scratch GPU renderer

`DayNightRenderer.java` is an in-progress, from-scratch GPU renderer (an original
pipeline, not derived from RuneLite's GPU plugin). It is **not** part of the published
"GPU Day Night" plugin: it lives outside `src/` so it is never compiled, checkstyled,
or loaded by the released plugin. It is kept here so the work isn't lost and can be
resumed later.
